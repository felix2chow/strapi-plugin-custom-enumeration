改了一下 strapi-plugin-multi-select，改成单选了

```
# Using Yarn
yarn add strapi-plugin-custom-enumeration

# Or using NPM
npm install strapi-plugin-custom-enumeration
```

Then, you'll need to build your admin panel:

```
# Using Yarn
yarn build

# Or using NPM
npm run build
```
